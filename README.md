# README

## This application is a mini API.
## It's a challenge for some job application.
### **Uses a brand new gem created by me called fedex**. Together API & GEM get shipping quotes.

* Ruby version:
```ruby-3.1.2```

* Database creation:
```rails db:drop db:create```

* Database initialization:
```rails db:migrate db:seed```

* How to run the test suite:
```rails test```

* How to run the application:
```rails s``` then [http://localhost:3000](http://localhost:3000)
